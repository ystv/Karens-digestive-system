# YSTV Karen's Digestive System

This repository houses everything needed to turn a computer into an dedicated, automatic ingest machine to make ingesting footage, and otherwise, easier.

## Installing

You will need [Python 3.0](https://www.python.org/) onwards and a computer running linux. This program was developed in [Ubuntu 22.04.1 LTS](https://ubuntu.com/) (Other linux distros are untested).

Clone this repo inside a chosen directory:

```shell
$ git clone https://github.com/ystv/Karens-digestive-system.git 
```

## Running

To start the program, execute the file `main.py`:

```shell
$ python3 main.py
```
*Run this inside the directory containing the file*
