# YSTV Karen's Digestive System

This repo houses everything needed to turn a computer into an dedicated, automatic ingest machine to make ingesting footage, and otherwise, easier.

## Installing

You will need [Python 3.0](https://www.python.org/) onwards and a computer running [Ubuntu](https://ubuntu.com/) (Other operating systems are untested - use at own risk).

Clone this repo inside a chosen directory:

```shell
$ git clone https://github.com/ystv/Karens-digestive-system.git 
```

## Running

To start the program, execute the file `index.py`:

```shell
$ python3 index.py
```
*Run this inside the directory containing the file*
